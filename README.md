# Behavioral Detection Engineering

Behavioral detection engine for identifying malware Command-and-Control (C2) beaconing and DNS-based Domain Generation Algorithm (DGA) activity using enterprise telemetry.

## Features

- Beacon Detection
- NXDOMAIN Burst Detection
- Cross-Telemetry Correlation
- Statistical Behavioral Analysis

## Dataset

- flows.csv
- dns.csv

## Requirements

Python 3.11+

Install dependencies

pip install -r requirements.txt

## Run

python src/program1_detection.py

## Output

- beacon_results.csv
- nxdomain_results.csv
